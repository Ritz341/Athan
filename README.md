# Islamic Smart Clock (Athan)

Lightweight portable Islamic smart clock for desktops. Includes Azan audio and simple settings for prayer times.

## Features
- HTML/CSS/JS single-folder app — open `index.html` to run locally
- Configurable settings in `settings.json`
- Assets (audio, images, fonts) included

## Quick start
1. Clone the repo:

```bash
git clone https://github.com/Ritz341/Athan.git
cd Athan
```

2. Open `index.html` in your browser (no server required). For development, you can use a simple static server:

```bash
npm install -g live-server
live-server
```

3. Configure settings in `settings.json` and place custom audio in `assets/audio/`.

## Packaging for distribution
- Zip the project folder to share a portable copy.
